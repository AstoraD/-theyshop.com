Clases multiples

Los elementos HTML pueden pertenecer a más de una clase. Para definir varias clases, separamos los nombres de las clases con un espacio. El elemento se ajustará al estilo de todas las clases especificadas.

Por ejemplo, todos los h2 pertenecen a la clase "tablita". Adicionalmente, London también pertenece a la clase "main", que se encarga de alinear el texto.

Uso de la clase Attribute en JavaScript

JavaScript también puede utilizar el nombre de la clase para realizar determinadas tareas para elementos específicos. JavaScript puede acceder a elementos con un nombre de clase específico con el método "getElementsByClassName()"
