Elementos semáticos HTML

Los elementos semánticos son etiquetas HTML que transmiten el significado o propósito del contenido que encapsulan, es decir, proporcionan un contexto más claro sobre el tipo de contenido que contienen

Elementos "figure" y "figcaption"

La etiqueta "figure" especifica contenido autónomo, como ilustraciones, diagramas, fotografías, listas de códigos, etc. La etiqueta "figcaption" define el título de un elemento con la etiqueta "figure".
