Ejemplo de Codificación de Caracteres en HTML

Este documento está codificado con UTF-8. Puedes cambiar la línea <meta charset="..."> para probar con otras codificaciones.

Comparación de Caracteres (Posiciones Comunes)
Código Descripción Carácter
128 Euro (€) €
133 Elipsis (…) …
145 Comilla simple izquierda (‘) ‘
147 Comilla doble izquierda (“) “
150 Guion largo (–) –
169 © Copyright ©
174 ® Registered ®
255 ÿ (y con diéresis) ÿ

Nota: Algunos caracteres podrían no verse correctamente si el navegador no usa la codificación adecuada.

Para probar, cambia el valor de la etiqueta <meta charset="..."> por:

  • UTF-8 (recomendado)
  • Windows-1252
  • ISO-8859-1
